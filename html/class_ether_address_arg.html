Detailed Description

Parser class for Ethernet addresses.

This is the default parser for objects of EtherAddress type. For 6-byte arrays like "click_ether::ether_shost" and "click_ether::ether_dhost", you must pass an EtherAddressArg() explicitly:

struct click_ether ethh;
... Args(...) ...
.read_mp("SRC", EtherAddressArg(), ethh.ether_shost)
...
