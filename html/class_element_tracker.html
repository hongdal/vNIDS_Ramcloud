Detailed Description

Base class for router configuration visitors that collect elements.

ElementTracker is a type of RouterVisitor used to traverse the router configuration graph and collect matching elements. A subclass's visit() function will call insert() to store elements that match. ElementTracker objects are usually passed to the Router::visit_downstream() and Router::visit_upstream() functions.
