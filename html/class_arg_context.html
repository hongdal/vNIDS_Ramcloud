Detailed Description

Argument context class.

The ArgContext class encapsulates state useful for parsing arguments: an element context and an ErrorHandler for reporting parse errors.

Args is derived from ArgContext. Some parser functions take an ArgContext reference rather than an Args reference. This clarifies that the parser function doesn't modify Args internals. Also, ArgContext objects are smaller and quicker to construct than Args objects.
