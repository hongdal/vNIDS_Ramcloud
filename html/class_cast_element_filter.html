Detailed Description

Cast-based filter predicate for elements.

Deprecated:
This class is deprecated. Use ElementCastTracker instead.

This ElementFilter type matches elements that match a given cast type. The constructor's string is passed to Element::cast(); the filter matches an element iff the result is non-null.
